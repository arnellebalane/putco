# putco

Post highlighted text to [putco.de][1], and copy it's url to the clipboard.

![putco demo][2]


## Installation

```bash
apm install putco
```

or install it from `Settings > Install` and look for `putco`.


## Usage

In the `Command Palette`, search for `Putco: Post Selected Text`, or press
`Ctrl-Alt-Shift-P`.
